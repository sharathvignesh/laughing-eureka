## Steps to run

* Clone this repository.

```bash
$ git clone https://github.com/sharathvignesh/laughing-eureka.git
```
* Install npm dependencies.

```bash
$ cd laughing-eureka
$ npm install
```

* Run the `index.js` file.

```bash
$ npm start
```

## Test
This project uses [mocha](https://github.com/mochajs/mocha) for testing.

```bash
$ npm test
```
